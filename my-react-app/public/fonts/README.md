# Satoshi Font Files

Please place your Satoshi font files in this directory:

## Required Files:
- `Satoshi-Variable.woff2` (or `.woff`)
- `Satoshi-VariableItalic.woff2` (or `.woff`)

## Optional Fallback Files:
- `Satoshi-Regular.woff2` (or `.woff`)
- `Satoshi-Medium.woff2` (or `.woff`)
- `Satoshi-Bold.woff2` (or `.woff`)

## Download Sources:
- [FOSSFonts](https://fossfonts.com/satoshi)
- [FontChase](https://fontchase.com/satoshi-font-family/)
- [Dafont Free](https://www.dafontfree.io/satoshi-font/)
